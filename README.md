# Hydrogen Shopify Store Demo

## Getting started

**Requirements:**

- Node v14+
- Yarn

```bash
yarn
yarn dev
```

## Building for production

```bash
yarn build
```

Then, you can run a local `server.js` using the production build with:

```bash
yarn serve
```
